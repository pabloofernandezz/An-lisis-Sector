# Ranking Sectorial Multifactorial - Motor Cuantitativo
Este proyecto analiza diferentes sectores del mercado utilizando un enfoque multifactorial para identificar oportunidades de inversión.

## 📊 Descripción
El objetivo de este script es procesar datos financieros y generar un ranking basado en factores cuantitativos. 
*Desarrollado por Pablo y J.*

## 🛠️ Tecnologías utilizadas
* **Python** (Google Colab)
* **Pandas** para manejo de datos
* **YFinance** para obtención de datos de mercado
* **Matplotlib** para visualización

## 📂 Contenido del Repositorio
* `Análisis_Sector.ipynb`: Código principal del motor cuantitativo.
* `Memoria_Proyecto.pdf`: Documentación detallada con las tesis de inversión y conclusiones.
